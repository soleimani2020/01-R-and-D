# Matrix-Vector Dot Product

**Difficulty:** Easy  
**Topic:** Linear Algebra

---

## 🧩 Problem Statement

Write a Python function that computes the **dot product of a matrix and a vector**.

- Return a list representing the resulting vector **if the operation is valid**.
- Return **-1** if the matrix and vector dimensions are incompatible.

A matrix (list of lists) can be dotted with a vector (list) **only if** the number of columns in the matrix equals the length of the vector.

---

## 🔹 Examples

**Example 1:**  

Input:
matrix = [[1, 2, 3],
[4, 5, 6]]
vector = [7, 8, 9]

Output: [50, 122]
Explanation:
[17 + 28 + 39 = 50,
47 + 58 + 69 = 122]
